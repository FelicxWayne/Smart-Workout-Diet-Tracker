# Smart Workout & Diet Tracker (Console App)

A Java-based menu-driven console application that intelligently suggests workouts
using Data Structures like HashMap and PriorityQueue.

## Features
- Add meals (calories & protein tracking)
- Add workouts (muscle groups)
- Smart workout suggestion based on recovery gap
- Daily calorie & protein summary
- Robust input handling

## Tech Used
- Java
- HashMap, PriorityQueue
- OOP principles

## How to Run
```bash
git clone https://github.com/FelicxWayne/Smart-Workout-Diet-Tracker.git
cd Smart-Workout-Diet-Tracker
javac *.java
java Main
